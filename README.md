# twitter-election

## Setup

1. Clone this repo
2. Create an app in the twitter app area: https://apps.twitter.com/
3. Create a file called `secret.json` in the root of the project similar to this:
```
{
    "consumer_key":"",
    "consumer_secret": "",
    "access_token": "",
    "access_token_secret": ""
}
```
4. Run `npm start`
